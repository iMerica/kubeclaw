# Support

## Where to Ask for Help

- **Usage questions / setup help**: open a GitHub Discussion or issue
- **Bug reports**: use the bug issue template
- **Feature requests**: use the feature request template
- **Security reports**: follow [SECURITY.md](SECURITY.md) and report privately

## What to Include

To speed up support, include:
- chart version
- Kubernetes version
- Helm command/values used (redact secrets)
- relevant logs/events
- expected vs actual behavior

## Response Expectations

KubeClaw is maintained on a best-effort basis.
Critical production-impacting bugs and security issues are prioritized first.

## Commercial / Enterprise Inquiries

For enterprise capabilities and related questions, see [kubeclaw.ai](https://kubeclaw.ai).
